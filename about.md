# Subtick Inputs API

API for developers who want subtick input handling. This mod does not do anything on its own and is intended to be used by other mods such as Superb Input Precision.

<mod:chizz.superb-input-precision>

## Incompatibilities

<mod:chizz.velocity-unrounding>

- The API exposes a Velocity Unrounding toggle already so there is no need for the standalone mod.

<mod:nytelyte.wave_trail_drag_fix>
<mod:nytelyte.wave_trail_draw_fix>

- Some of the many issues with wave trail seem to be linked with these mods, hopefully I can make them compatible in the future.
